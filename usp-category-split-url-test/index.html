US Patriot Category Split URL Test

For this test I created a mid-level category page, to go in between the CLP and PLP.

The hypothesis for this test was that if users were able to drill down to the boot they wanted before being presented with the full PLP, they'd be more likely to add a pair of boots to their cart. That hypothesis was proven false by this test. The custom-coded carousels were made using the Slick slider plugin.
